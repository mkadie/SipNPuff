# Prototype Photos

First article prototype of the T-Rex Sip-N-Puff HID electronics assembly.

These photos document the initial build of the Raspberry Pi Pico-based sip and puff human interface device, including the MPX5010DP pressure sensor, PC817 optocoupler outputs for the Xbox Adaptive Controller, and optional I2C display modules.

## Contents

| File | Description |
|------|-------------|
| PCB front view | Top side of the assembled board showing the Pico, pressure sensor, and supporting components |
| PCB back/rotated | Reverse angle showing board layout |
| PCB with TFT display | Assembly with TFT display module attached via I2C |
| PCB with OLED and tubing | Assembly with OLED display and sip/puff tubing connected to the MPX5010DP |
| PCB underside | Solder joint quality on the bottom of the board |
| Full assembly | Complete build with sip tube mounted to the board |

## Notes

- This is a first article prototype — form factor and layout are subject to change as the enclosure design matures.
- See the `3d/` directory for the initial mechanical model.
- See the project root for full BOM, schematic, and firmware.
